# bicycle-recommendation
Side project

# How to setup this project

1. Install XAMPP, prefer with PHP 5
2. Clone this project
3. Copy and paste this project your web server directory
	If using XAMPP usually the directory is : C:\xampp\htdocs
4. Start Apache server and MySQL server on XAMPP
5. Open PHPMYADMIN : localhost/phpmyadmin
6. Create new database name : bicycledb
7. Import sql file from project folder (bicycledb.sql)
